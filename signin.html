Made for Mobile and PC

1. We used to design seperately for mobile site and PC, now they share the same design. 2. With the design solution for both mobile site and PC, two groups of developers (developers of mobile site and developers of PC) collabrate together and push the whole project online.









Bring auto fill mail suffix into the app

There will be mail suffix appear automaticlly when user type "@"，and it's related to countries. For instances, @yandex.com will be a choice for Russian user, while @gmail.com should always be a selection for US user.







Clear Definition of touch Points

User registration and login behavior occurs on various occasions，it can be at home or on a crowded metro. Thus when user input their password, it should by default be unvisible unless the user ture on the visible option.









Smart verification

When it's detected that the user had already registered, a popup will appear to notice. And when user switch to login, all the email information will stay there so that it's not nessisary for user to input again.





Embrace Tech
